FAQ

What games are supported?

SwagWare supports most games that don’t use custom player models.

Will I get banned?

SwagWare can be used on any game, but anti-cheats may detect certain features.

What makes SwagWare unique?

  • Stream-proof
  • Silent aim
  • Windows 11 support

What do I need to run SwagWare?

Any decent PC on Windows 10/11 (up to version 22H2) will work.

Custom Lua Environment Documentation (swag API)
SWAGWARE runs Lua 5.4.8 with a custom library called swag.
This scripting environment is unique to SWAGWARE and is not related to Roblox’s Lua environment in any way.

Global Namespace: swag

All SWAGWARE scripting functions are accessed through the swag global. Use these to interact with in-game objects and player data.

  • swag.get_datamodel()
    This function returns the root object of the game. If you’ve used Lua before, think of it like the game object in other engines. It’s essential for accessing things like the workspace or player list.
  • swag.find_first_child(parent, name)
    Use this to search for a child object by name under any instance. Returns the first match or nil if not found.
  • swag.get_children(instance)
    Returns a table (array) of all children under the given instance. Useful for iterating through groups of objects.
  • swag.get_local_player()
    Returns the memory address of the local player. (Note: This is for internal/C++ use, not for scripting.)
  • swag.get_health()
    Returns the local player’s current health value.
  • swag.get_max_health()
    Returns the local player’s maximum health value.
  • swag.get_walkspeed()
    Returns the local player’s current walkspeed.
  • swag.set_walkspeed(speed)
    Sets the local player’s walkspeed to the value you provide.
  • swag.get_part_pos(partName)
    Returns the position (x, y, z) of a part (by name) on the local player.

Instance Object Methods

  • instance:getName()
    Returns the name of the instance or child.
  • instance:getClassName()
    Returns the class type of the instance (like "Part" or "Model").

Example Script with Comments


-- Get the root object of the game
local game = swag.get_datamodel()

-- Try to get the "Workspace" child from the root
local workspace = swag.find_first_child(game, "Workspace")

-- Safety check in case it's not found
if not workspace then
    print("Workspace not found!")
    return
end

-- Get all children of Workspace
local children = swag.get_children(workspace)

print("Children of Workspace:")

-- Loop over each child and print its name and class
for _, child in ipairs(children) do
    local name = child:getName()
    local class = child:getClassName()
    print("- " .. name .. " [" .. class .. "]")
end
