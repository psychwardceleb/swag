Initial Setup Guide

1. Make Sure Your Windows Account Has Administrator Privileges

To run the loader properly, your Windows account must be an administrator. This ensures it can execute without prompting for another username or password.

2. SwagWare Anti-Virus Exclusions

To avoid issues with your anti-virus, you need to exclude the following folders:

  • The folder where your loader.exe is located.
  • The %appdata%\swagware folder.

FAQ

What games are supported?

Almost all of them, really. SwagWare supports any games that don’t have custom player models. We are working on adding support for the games that do.

Will I get banned?

You can use SwagWare on any game freely, but games with advanced anti-cheats may detect certain features or unusual aiming behavior.

What makes SwagWare unique?

  • Stream-proof, bypasses Discord screen shares.
  • Well-coded silent aim.
  • Windows 11 support.

What do I need to run SwagWare?

You don’t need a NASA supercomputer to run SwagWare—a solid PC will do just fine! Just make sure you're on Windows 10 or Windows 11 (up to version 22H2).
